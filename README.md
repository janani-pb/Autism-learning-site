# 🌟 Link to Lights

**An interactive, level-based learning platform designed for autistic children aged 5–9.**

Link to Lights helps young learners on the autism spectrum build foundational skills through structured, engaging, and low-stimulation-friendly digital activities. The platform breaks learning into progressive levels so children can build confidence and stay engaged at their own pace.

---

## 📖 About the Project

Children with autism often benefit from predictable structure, visual clarity, and gradual skill progression. Link to Lights was built around these principles — using simple layouts, clear feedback, and a tiered level system to keep young learners engaged without overwhelming them.

---

## ✨ Key Features

- 🎯 **3-Level Progressive Learning System** — each level increases in complexity, letting children advance as their skills and confidence grow
- 🖱️ **Simple, Distraction-Free Interface** — designed with sensory sensitivities in mind
- 🔄 **Interactive Activities** — React-driven components for real-time feedback and engagement
- 💾 **Backend Data Handling** — PHP + XAMPP (MySQL) used to manage learner progress and content
- 🎨 **Clean, Calming Visual Design** — HTML/CSS layout built for readability and reduced visual clutter

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend UI | HTML, CSS, React |
| Interactivity | React Components |

---

## 🎯 Target Users

Designed specifically for **autistic children aged 5–9**, with input from the structure of special-education learning approaches — focusing on repetition, positive reinforcement, and gradual skill-building across levels.

---

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/janani-pb/link-to-lights.git
   ```
2. Install XAMPP and start Apache + MySQL
3. Move the project folder into `htdocs`
4. Import the database (if `.sql` file is included in `/database`)
5. Run the frontend:
   ```bash
   npm install
   npm start
   ```
6. Open `http://localhost/link-to-lights` in your browser
